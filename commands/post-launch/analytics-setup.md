# 🔀 Command: analytics-setup

> Configures product analytics and event tracking.

## Instructions for Claude

1. Read `requirements.md` and `mvp-definition.md` for success metrics
2. Set up analytics tool (from stack-selection.md)
3. Define and implement tracking for:
   - **Acquisition:** How users arrive (UTM params, referrer)
   - **Activation:** First meaningful action (onboarding complete, first [core action])
   - **Engagement:** Daily/weekly usage, feature adoption
   - **Retention:** Return rate, churn signals
   - **Revenue:** Conversion, plan upgrades, LTV (if applicable)
4. Create dashboard with key metrics
5. Verify events fire correctly in staging before production
