# 📝 Command: decision-log

> Records architectural and technical decisions (ADR format).

## Template

For each significant decision, add an entry:

---

### ADR-[number]: [title]

**Date:** 
**Status:** proposed / accepted / deprecated / superseded

**Context:**
<!-- What is the situation? What problem are we solving? -->

**Decision:**
<!-- What did we decide? -->

**Alternatives considered:**
| Option | Pros | Cons |
|--------|------|------|
|        |      |      |

**Consequences:**
<!-- What are the implications? What do we gain/lose? -->

---

## When to log

- Choosing a major library or framework
- Deciding on an architectural pattern
- Choosing between build vs buy
- Changing a previous decision
- Any choice that would confuse a future developer without context
