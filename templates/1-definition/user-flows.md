# 🔀 User Flows

> Document the critical paths a user takes. Claude can generate diagrams from these.

## Flow 1: Onboarding (first-time user)

```
1. User lands on homepage
2. Clicks "Sign up"
3. Fills registration form (email, password)
4. Receives verification email
5. Clicks verification link
6. Lands on onboarding wizard
7. Completes profile setup
8. Sees dashboard for the first time
```

**Happy path duration:** ~3 minutes
**Drop-off risks:** Step 4 (email not received), Step 6 (skips onboarding)
